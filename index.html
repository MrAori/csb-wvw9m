Project Synthestia

About this project

This project is to show a prototype piano with an audio visualiser to feel how loud, quiet, low or high a type of sound could make a sphere big or small.

How does this work?

Click/tap on any key of the corresponding note, Hold anywhere not on the keys to rotate the camera around and to change the pitch, the settings should be in the GUI settings; the default is set to 3.

NOTICE

Keep in mind this is a prototype so do not expect things to be completely polished, not every key on the piano is mapped so bear in mind. It's also recommended to try this on computers, it may work on mobiles but performance may vary. This is for testing purposes only.
